## Overview

**Coupons** (in the left nav) are the discount codes customers redeem against a
subscription. Each coupon is either a **percent-off** or a **fixed-amount-off**
discount, applied for a chosen **duration**.

The list shows every coupon with these columns:

| Column | What it shows |
| --- | --- |
| Coupon Code | The code customers enter |
| Discount | e.g. `25%` or `$10.00` |
| Duration | Once, Forever, or *For N months* |
| Redemptions | Times used (and the cap, if set) |
| Status | **Active** or **Inactive** |

Search by code and filter with the **All / Active / Inactive** tabs. Each row has
a quick **Deactivate** / **Reactivate** action on the right.

## Create a coupon

<Steps>
  <Step title="Open Create coupon">
    Click **Create coupon** to open the side panel.
  </Step>
  <Step title="Set the code">
    Type a **Coupon code** (e.g. `SUMMER25OFF`) or click **Generate** for a random
    10-character code.
  </Step>
  <Step title="Choose the discount">
    Pick a **Discount type** and enter a **Discount value**:

    - **Percent off** — a whole-number percentage (the field shows a `%`).
    - **Amount off** — a currency amount (the field shows a `$`).
  </Step>
  <Step title="Set duration & limits">
    - **Duration** — **Once**, **Forever**, or **Limited time (repeating)**.
      Repeating reveals a **Duration in months** field.
    - **Max redemptions** — optional; leave blank for unlimited.
    - **Status** — toggle the coupon active or inactive on creation.

    Click **Create coupon**.
  </Step>
</Steps>

<Info>
Enter amount-off discounts in whole currency units — `25` means **$25 off**, not
25 cents. Percent discounts are plain numbers.
</Info>

## The coupon detail panel

Click a row to open the coupon's detail sheet. It shows the code with its
discount badge, the **Duration**, and the **Created** date. When the coupon has
a **redemption cap**, a progress bar shows how much of the cap has been used;
otherwise it shows the **times redeemed** count.

## Activate & deactivate

Use the row action (or reactivate directly — it's low-risk and skips
confirmation) to flip a coupon's status. Deactivation asks you to confirm.

<Warning>
Deactivating a coupon stops **new** subscriptions from redeeming the code.
Customers already using it **keep their discount**. You can reactivate the code
later.
</Warning>
