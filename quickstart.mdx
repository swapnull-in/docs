## Installation

Choose your preferred setup method:

<Tabs>
  <Tab title="Docker (Recommended)">
    ```bash
    # Clone the repository
    git clone https://github.com/recur-so/recurso.git
    cd recurso

    # Start with Docker Compose
    docker-compose up -d

    # API available at http://localhost:8080
    ```
  </Tab>
  <Tab title="From Source">
    ```bash
    # Clone the repository
    git clone https://github.com/recur-so/recurso.git
    cd recurso

    # Install dependencies
    go mod download

    # Set up environment
    cp .env.example .env

    # Run migrations
    make migrate

    # Start the server
    make run
    ```
  </Tab>
  <Tab title="SDK Only">
    ```bash
    # If using Recurso Cloud or existing deployment
    npm install @recurso/sdk
    # or
    pip install recurso
    # or
    go get github.com/recur-so/recurso-go
    ```
  </Tab>
</Tabs>

## Configuration

Set these environment variables:

```bash .env
# Database
DATABASE_URL=postgres://user:pass@localhost:5432/recurso

# API
PORT=8080
API_KEY_SECRET=your-secret-key

# Payment Gateways (optional)
RAZORPAY_KEY_ID=rzp_test_xxx
RAZORPAY_KEY_SECRET=xxx
STRIPE_SECRET_KEY=sk_test_xxx

# Email (optional)
SMTP_HOST=smtp.example.com
SMTP_PORT=587
SMTP_USER=noreply@example.com
SMTP_PASSWORD=xxx
```

## Initialize the SDK

<CodeGroup>
```typescript TypeScript
import Recurso from '@recurso/sdk';

const recurso = new Recurso({
  apiKey: process.env.RECURSO_API_KEY,
  baseUrl: 'http://localhost:8080' // or your deployment URL
});
```

```python Python
from recurso import Recurso

recurso = Recurso(
    api_key=os.environ['RECURSO_API_KEY'],
    base_url='http://localhost:8080'
)
```

```go Go
import "github.com/recur-so/recurso-go"

client := recurso.NewClient(
    os.Getenv("RECURSO_API_KEY"),
    recurso.WithBaseURL("http://localhost:8080"),
)
```
</CodeGroup>

## Create Your First Plan

<Steps>
  <Step title="Create a pricing plan">
    ```typescript
    const plan = await recurso.plans.create({
      name: 'Pro Plan',
      description: 'For growing teams',
      interval: 'month',
      prices: [
        { amount: 4999, currency: 'INR' }, // â‚¹49.99/month
        { amount: 999, currency: 'USD' }   // $9.99/month
      ]
    });
    ```
  </Step>
  
  <Step title="Add a customer">
    ```typescript
    const customer = await recurso.customers.create({
      name: 'Acme Corp',
      email: 'billing@acme.com',
      metadata: {
        company_size: 'mid-market'
      }
    });
    ```
  </Step>
  
  <Step title="Start a subscription">
    ```typescript
    const subscription = await recurso.subscriptions.create({
      customer_id: customer.id,
      plan_id: plan.id,
      payment_gateway: 'razorpay' // or 'stripe'
    });

    // Redirect customer to complete payment
    console.log(subscription.checkout_url);
    ```
  </Step>
  
  <Step title="Handle webhooks">
    ```typescript
    app.post('/webhooks/recurso', (req, res) => {
      const event = recurso.webhooks.verify(req.body, req.headers);
      
      switch (event.type) {
        case 'subscription.activated':
          // Grant access to your product
          break;
        case 'payment.succeeded':
          // Payment received
          break;
        case 'subscription.cancelled':
          // Revoke access
          break;
      }
      
      res.sendStatus(200);
    });
    ```
  </Step>
</Steps>
