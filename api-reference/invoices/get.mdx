---
title: "Get Invoice"
openapi: "GET /v1/invoices/{id}"
description: "Retrieve a specific invoice"
---

## Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | The invoice ID |

## Example Request

```bash
curl https://api.recurso.dev/v1/invoices/inv_001 \
  -H "Authorization: Bearer $API_KEY"
```
