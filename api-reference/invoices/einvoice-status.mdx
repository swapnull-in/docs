---
title: "Get E-Invoice Status"
openapi: "GET /v1/invoices/{id}/einvoice"
description: "Retrieve the e-invoice status for an invoice"
---

## Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | The invoice ID |

## Example Request

```bash
curl https://billing.example.com/v1/invoices/inv_001/einvoice \
  -H "Authorization: Bearer $API_KEY"
```

## Response

```json
{
  "id": "einv_8mw3p1",
  "object": "einvoice",
  "invoice_id": "inv_001",
  "status": "generated",
  "irn": "a1b2c3d4e5f6a1b2c3d4e5f6a1b2c3d4e5f6a1b2c3d4e5f6a1b2c3d4e5f6a1b2",
  "ack_number": "112310037956284",
  "ack_date": "2024-01-15T12:30:00Z",
  "signed_qr_code": "eyJhbGciOiJSUzI1NiIsInR5cCI6...",
  "error": null,
  "created_at": "2024-01-15T12:00:00Z",
  "updated_at": "2024-01-15T12:30:00Z"
}
```

<Info>
E-invoicing is required for businesses in India with turnover above the threshold set by GST regulations. The `irn` (Invoice Reference Number) is a unique identifier issued by the government e-invoice portal.
</Info>

### Status Values

| Status | Description |
|--------|-------------|
| `pending` | E-invoice generation has been queued |
| `generated` | IRN successfully obtained from the portal |
| `failed` | Generation failed — check the `error` field |
| `cancelled` | E-invoice has been cancelled |
