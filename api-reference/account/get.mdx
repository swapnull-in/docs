---
title: "Get Account"
openapi: "GET /v1/account"
description: "Retrieve the details and settings of your Recurso account."
---

## Query Parameters

This endpoint does not accept any query parameters.

## Example Request

```bash
curl -X GET "https://api.recurso.io/v1/account" \
  -H "Authorization: Bearer rsk_live_9f8a7b6c5d4e3f2a1b0c" \
  -H "Content-Type: application/json"
```

### Plan Values

| Plan         | Description                                |
|--------------|--------------------------------------------|
| `starter`    | Up to 500 active subscriptions             |
| `growth`     | Up to 10,000 active subscriptions          |
| `scale`      | Up to 100,000 active subscriptions         |
| `enterprise` | Unlimited subscriptions, dedicated support |
