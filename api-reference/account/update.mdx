---
title: "Update Account"
openapi: "PUT /v1/account"
description: "Update your Recurso account settings."
---

## Body Parameters

| Parameter          | Type   | Required | Description                                                            |
|--------------------|--------|----------|------------------------------------------------------------------------|
| `name`             | string | No       | The display name for your account.                                     |
| `email`            | string | No       | Primary billing contact email address.                                 |
| `timezone`         | string | No       | IANA timezone identifier (e.g. `America/New_York`, `Europe/London`).   |
| `default_currency` | string | No       | Three-letter ISO 4217 currency code (e.g. `usd`, `eur`, `gbp`).      |
| `webhook_url`      | string | No       | URL where Recurso sends webhook event payloads.                        |

Only the fields you include in the request body will be updated. Omitted fields remain unchanged.

## Example Request

```bash
curl -X PUT "https://api.recurso.io/v1/account" \
  -H "Authorization: Bearer rsk_live_9f8a7b6c5d4e3f2a1b0c" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Acme Corp International",
    "timezone": "Europe/London",
    "default_currency": "gbp",
    "webhook_url": "https://api.acmecorp.com/webhooks/recurso/v2"
  }'
```
