---
title: "Initiate Checkout Payment"
openapi: "POST /checkout/{id}/pay"
description: "Initiate payment for a checkout session"
---

<Note>
This is a public endpoint. No authentication is required.
</Note>

## Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | The invoice ID |

## Request Body

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `payment_method` | string | No | Preferred payment method (e.g. `upi`, `card`, `netbanking`) |

## Example Request

```bash
curl -X POST https://billing.example.com/checkout/inv_001/pay \
  -H "Content-Type: application/json" \
  -d '{
    "payment_method": "upi"
  }'
```

## Response

```json
{
  "order_id": "order_r4z8k1m2n3",
  "gateway": "razorpay",
  "amount": 5899,
  "currency": "INR",
  "checkout_url": "https://api.razorpay.com/v1/checkout/order_r4z8k1m2n3"
}
```

<Warning>
If the invoice has already been paid or is in a `void` status, this endpoint returns a `400` error. Always check the invoice status before initiating payment.
</Warning>
