---
title: "Get Organization"
openapi: "GET /v1/organizations/{id}"
description: "Retrieve details of a specific organization by its ID."
---

## Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `id` | string | Yes | The unique identifier of the organization (e.g. `org_4f3e2d1c0b9a8765`). |

## Example Request

```bash
curl https://api.recurso.dev/v1/organizations/org_4f3e2d1c0b9a8765 \
  -H "Authorization: Bearer rsk_live_xxx"
```
