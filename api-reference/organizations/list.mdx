---
title: "List Organizations"
openapi: "GET /v1/organizations"
description: "Retrieve all organizations for the authenticated account."
---

## Example Request

```bash
curl https://api.recurso.dev/v1/organizations \
  -H "Authorization: Bearer rsk_live_xxx"
```

## Response

Returns a list of organization objects.
