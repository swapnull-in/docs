---
title: "Create Coupon"
openapi: "POST /v1/coupons"
description: "Create a new discount coupon"
---

## Request Body

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `code` | string | Yes | Unique coupon code |
| `discount_type` | string | Yes | `percentage` or `fixed` |
| `discount_value` | number | Yes | Discount amount (percentage or fixed amount in smallest currency unit) |
| `currency` | string | No | Three-letter ISO currency code (required for `fixed` discount type) |
| `duration` | string | Yes | `once`, `repeating`, or `forever` |
| `duration_in_months` | integer | No | Number of months the coupon applies (required when duration is `repeating`) |
| `max_redemptions` | integer | No | Maximum number of times this coupon can be redeemed |

## Example Request

```bash
curl -X POST https://billing.example.com/v1/coupons \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "code": "SUMMER25",
    "discount_type": "percentage",
    "discount_value": 25,
    "duration": "repeating",
    "duration_in_months": 3,
    "max_redemptions": 500
  }'
```
