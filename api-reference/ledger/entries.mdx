---
title: "List Ledger Entries"
openapi: "GET /v1/ledger/entries"
description: "Retrieve ledger entries with optional filters for account, date range, and pagination."
---

## Query Parameters

| Parameter    | Type   | Required | Description                                                         |
|--------------|--------|----------|---------------------------------------------------------------------|
| `account_id` | string | No       | Filter entries to a specific ledger account (e.g., `acct_1aB2CdEfGh`). |
| `start_date` | string | No       | ISO 8601 date for the beginning of the range (inclusive).           |
| `end_date`   | string | No       | ISO 8601 date for the end of the range (inclusive).                 |
| `limit`      | int    | No       | Maximum number of entries to return. Defaults to `50`, max `200`.   |
| `offset`     | int    | No       | Number of entries to skip for pagination. Defaults to `0`.          |

Each ledger entry represents a double-entry transaction with one or more debit and credit movements. All amounts are expressed in the smallest currency unit.

## Example Request

```bash
curl -X GET "https://api.recurso.io/v1/ledger/entries?account_id=acct_1aB2CdEfGh&start_date=2026-06-01&end_date=2026-06-30&limit=2" \
  -H "Authorization: Bearer sk_live_26PHem9AhJZvU..." \
  -H "Content-Type: application/json"
```
