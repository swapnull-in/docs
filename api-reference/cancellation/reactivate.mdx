---
title: "Reactivate Subscription"
openapi: "POST /v1/subscriptions/{id}/reactivate"
description: "Reactivate a subscription that is scheduled for cancellation at the end of the current billing period."
---

## Path Parameters

| Parameter | Type   | Description                              |
|-----------|--------|------------------------------------------|
| `id`      | string | The unique identifier of the subscription (e.g., `sub_3jF6RtKmWp`). |

Reactivation is only possible when the subscription has `cancel_at_period_end: true` and the current billing period has not yet ended. If the subscription has already been fully cancelled, it cannot be reactivated through this endpoint.

## Example Request

```bash
curl -X POST https://api.recurso.io/v1/subscriptions/sub_3jF6RtKmWp/reactivate \
  -H "Authorization: Bearer sk_live_26PHem9AhJZvU..." \
  -H "Content-Type: application/json"
```
