---
title: "Start Cancel Flow Session"
openapi: "POST /v1/cancel-flows/sessions/start"
description: "Initiate a new cancellation flow session for a customer."
---

## Body Parameters

| Parameter         | Type   | Required | Description                                                        |
|-------------------|--------|----------|--------------------------------------------------------------------|
| `customer_id`     | string | Yes      | The unique identifier of the customer initiating cancellation.     |
| `subscription_id` | string | Yes      | The subscription the customer wants to cancel.                     |
| `flow_id`         | string | No       | Specific cancel flow to use. If omitted, the default flow is used. |
| `metadata`        | object | No       | Arbitrary key-value pairs to attach to the session.                |

## Example Request

```bash
curl -X POST "https://api.recurso.io/v1/cancel-flows/sessions/start" \
  -H "Authorization: Bearer rsk_live_9f8a7b6c5d4e3f2a1b0c" \
  -H "Content-Type: application/json" \
  -d '{
    "customer_id": "cust_4kTg8LpR2mXn",
    "subscription_id": "sub_7vWq3JsN9bYz",
    "flow_id": "cflow_01h8x3kv7m2p"
  }'
```
