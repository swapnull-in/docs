---
title: "List Cancel Flows"
openapi: "GET /v1/cancel-flows"
description: "Retrieve all cancel flows configured for your account."
---

## Query Parameters

| Parameter    | Type    | Required | Description                                              |
|--------------|---------|----------|----------------------------------------------------------|
| `limit`      | integer | No       | Maximum number of flows to return. Default `20`, max `100`. |
| `offset`     | integer | No       | Number of records to skip for pagination. Default `0`.   |
| `is_active`  | boolean | No       | Filter by active or inactive flows.                      |

## Example Request

```bash
curl -X GET "https://api.recurso.io/v1/cancel-flows?limit=10" \
  -H "Authorization: Bearer rsk_live_9f8a7b6c5d4e3f2a1b0c" \
  -H "Content-Type: application/json"
```
