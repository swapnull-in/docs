---
title: "Get Cancel Flow Statistics"
openapi: "GET /v1/cancel-flows/stats"
description: "Retrieve aggregated statistics for cancel flow sessions."
---

## Query Parameters

| Parameter    | Type   | Required | Description                                                        |
|--------------|--------|----------|--------------------------------------------------------------------|
| `flow_id`    | string | No       | Filter stats to a specific cancel flow. Returns all flows if omitted. |
| `start_date` | string | No       | Start of the date range in ISO 8601 format (e.g. `2026-01-01`).   |
| `end_date`   | string | No       | End of the date range in ISO 8601 format (e.g. `2026-06-30`).     |

## Example Request

```bash
curl -X GET "https://api.recurso.io/v1/cancel-flows/stats?start_date=2026-01-01&end_date=2026-06-23" \
  -H "Authorization: Bearer rsk_live_9f8a7b6c5d4e3f2a1b0c" \
  -H "Content-Type: application/json"
```
