---
title: "Get Customer"
openapi: "GET /v1/customers/{id}"
description: "Retrieve a specific customer by ID"
---

## Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | The customer ID |

## Example Request

```bash
curl https://api.recurso.dev/v1/customers/cust_abc123 \
  -H "Authorization: Bearer $API_KEY"
```
