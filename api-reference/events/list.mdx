---
title: "List Events"
openapi: "GET /v1/events"
description: "Retrieve a paginated list of webhook events"
---

## Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `type` | string | Filter by event type (e.g. `invoice.paid`) |
| `limit` | integer | Max results (default: 10, max: 100) |
| `offset` | integer | Pagination offset |

## Example Request

```bash
curl "https://api.recurso.dev/v1/events?type=invoice.paid&limit=5" \
  -H "Authorization: Bearer $API_KEY"
```
