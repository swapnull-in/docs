---
title: "Generate Advance Invoice"
openapi: "POST /v1/subscriptions/{id}/advance"
description: "Generate an advance invoice covering one or more future billing periods."
---

## Path Parameters

| Parameter | Type   | Description                              |
|-----------|--------|------------------------------------------|
| `id`      | string | The unique identifier of the subscription (e.g., `sub_3jF6RtKmWp`). |

## Request Body

| Parameter | Type | Required | Description                                                                 |
|-----------|------|----------|-----------------------------------------------------------------------------|
| `periods` | int  | Yes      | The number of future billing periods to invoice in advance. Minimum value is `1`. |

Generates a single invoice that covers the specified number of upcoming billing periods. Each period appears as a separate line item on the invoice. The invoice is created with `type: "advance"` and is immediately finalized.

## Example Request

```bash
curl -X POST https://api.recurso.io/v1/subscriptions/sub_3jF6RtKmWp/advance \
  -H "Authorization: Bearer sk_live_26PHem9AhJZvU..." \
  -H "Content-Type: application/json" \
  -d '{
    "periods": 3
  }'
```
