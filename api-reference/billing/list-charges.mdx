---
title: "List Pending Charges"
openapi: "GET /v1/subscriptions/{id}/charges"
description: "List all pending unbilled charges for a subscription."
---

## Path Parameters

| Parameter | Type   | Description                              |
|-----------|--------|------------------------------------------|
| `id`      | string | The unique identifier of the subscription (e.g., `sub_3jF6RtKmWp`). |

Returns all charges with `status: "pending"` that have not yet been invoiced. Once a charge is included on an invoice, it no longer appears in this list.

## Example Request

```bash
curl -X GET https://api.recurso.io/v1/subscriptions/sub_3jF6RtKmWp/charges \
  -H "Authorization: Bearer sk_live_26PHem9AhJZvU..." \
  -H "Content-Type: application/json"
```
