---
title: "Get Dunning Campaign"
openapi: "GET /v1/dunning-campaigns/{id}"
description: "Retrieve a single dunning campaign including all of its configured steps."
---

## Path Parameters

| Parameter | Type   | Required | Description                                       |
|-----------|--------|----------|---------------------------------------------------|
| `id`      | string | Yes      | The dunning campaign ID (`dc_` prefix).           |

## Example Request

```bash
curl -X GET "https://api.recurso.io/v1/dunning-campaigns/dc_8mKp3vRn2wXq" \
  -H "Authorization: Bearer rsk_live_9f8a7b6c5d4e3f2a1b0c" \
  -H "Content-Type: application/json"
```
