---
title: "List API Keys"
openapi: "GET /v1/developer/keys"
description: "Retrieve all API keys for your account. Key values are masked for security."
---

## Query Parameters

| Parameter     | Type   | Required | Description                                                  |
|---------------|--------|----------|--------------------------------------------------------------|
| `environment` | string | No       | Filter by environment. One of `test` or `live`.              |
| `limit`       | integer| No       | Maximum number of keys to return. Default `20`, max `100`.   |
| `offset`      | integer| No       | Number of records to skip for pagination. Default `0`.       |

## Example Request

```bash
curl -X GET "https://api.recurso.io/v1/developer/keys?environment=live" \
  -H "Authorization: Bearer rsk_live_9f8a7b6c5d4e3f2a1b0c" \
  -H "Content-Type: application/json"
```
