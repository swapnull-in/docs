---
title: "List Credit Notes"
openapi: "GET /v1/credit-notes"
description: "Retrieve a list of credit notes"
---

## Query Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `customer_id` | string | Filter by customer |
| `invoice_id` | string | Filter by invoice |
| `limit` | integer | Max results (default: 10) |
| `offset` | integer | Pagination offset |

## Example Request

```bash
curl "https://api.recurso.dev/v1/credit-notes?customer_id=cust_abc123&limit=10" \
  -H "Authorization: Bearer $API_KEY"
```
