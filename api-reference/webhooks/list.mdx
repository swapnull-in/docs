---
title: "List Webhooks"
openapi: "GET /v1/webhooks"
description: "Retrieve all webhook endpoints configured for your account."
---

## Query Parameters

| Parameter | Type    | Required | Description                                                     |
|-----------|---------|----------|-----------------------------------------------------------------|
| `limit`   | integer | No       | Maximum number of endpoints to return. Default `20`, max `100`. |
| `offset`  | integer | No       | Number of records to skip for pagination. Default `0`.          |
| `status`  | string  | No       | Filter by status: `active` or `inactive`.                       |

## Example Request

```bash
curl -X GET "https://billing.example.com/v1/webhooks?limit=10" \
  -H "Authorization: Bearer rsk_live_9f8a7b6c5d4e3f2a1b0c" \
  -H "Content-Type: application/json"
```
