---
title: "Get Quote"
openapi: "GET /v1/quotes/{id}"
description: "Retrieve a single quote by its unique identifier."
---

## Path Parameters

| Parameter | Type   | Description                              |
|-----------|--------|------------------------------------------|
| `id`      | string | The unique identifier of the quote (e.g., `qt_4kR7LmNpXs`). |

## Example Request

```bash
curl -X GET https://api.recurso.io/v1/quotes/qt_4kR7LmNpXs \
  -H "Authorization: Bearer sk_live_26PHem9AhJZvU..." \
  -H "Content-Type: application/json"
```
