---
title: "Send Quote"
openapi: "POST /v1/quotes/{id}/send"
description: "Send a quote to the customer via email."
---

## Path Parameters

| Parameter | Type   | Description                              |
|-----------|--------|------------------------------------------|
| `id`      | string | The unique identifier of the quote (e.g., `qt_4kR7LmNpXs`). |

Sending a quote transitions its status from `draft` to `sent`. The customer receives an email with a link to review and accept the quote. A quote can only be sent if it is currently in `draft` status.

## Example Request

```bash
curl -X POST https://api.recurso.io/v1/quotes/qt_4kR7LmNpXs/send \
  -H "Authorization: Bearer sk_live_26PHem9AhJZvU..." \
  -H "Content-Type: application/json"
```
