---
title: "Accept Quote"
openapi: "POST /v1/quotes/{id}/accept"
description: "Accept a sent quote on behalf of the customer."
---

## Path Parameters

| Parameter | Type   | Description                              |
|-----------|--------|------------------------------------------|
| `id`      | string | The unique identifier of the quote (e.g., `qt_4kR7LmNpXs`). |

Accepting a quote transitions its status from `sent` to `accepted`. Only quotes with `status: "sent"` that have not yet expired can be accepted. Once accepted, the quote can be converted into an invoice.

## Example Request

```bash
curl -X POST https://api.recurso.io/v1/quotes/qt_4kR7LmNpXs/accept \
  -H "Authorization: Bearer sk_live_26PHem9AhJZvU..." \
  -H "Content-Type: application/json"
```
