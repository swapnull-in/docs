---
title: "Get Plan"
openapi: "GET /v1/plans/{id}"
description: "Retrieve a specific plan by ID"
---

## Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | The plan ID |

## Example Request

```bash
curl https://api.recurso.dev/v1/plans/plan_pro \
  -H "Authorization: Bearer $API_KEY"
```

## Errors

| Code | Description |
|------|-------------|
| 404 | Plan not found |
