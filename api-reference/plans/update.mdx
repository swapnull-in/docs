---
title: "Update Plan"
openapi: "PUT /v1/plans/{id}"
description: "Update an existing pricing plan"
---

## Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | The plan ID |

## Request Body

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `name` | string | No | Display name for the plan |
| `description` | string | No | Plan description |
| `active` | boolean | No | Whether the plan is available for new subscriptions |
| `trial_days` | integer | No | Free trial period in days |
| `metadata` | object | No | Custom key-value data |

## Example Request

```bash
curl -X PUT https://api.recurso.dev/v1/plans/plan_pro_01 \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Pro Plan (Updated)",
    "description": "For growing teams — now with more features",
    "active": true,
    "trial_days": 21,
    "metadata": {
      "tier": "professional",
      "version": "2"
    }
  }'
```

## Response

```json
  "id": "plan_pro_01",
  "object": "plan",
  "name": "Pro Plan (Updated)",
  "description": "For growing teams — now with more features",
  "interval": "month",
  "interval_count": 1,
  "trial_days": 21,
  "active": true,
  "prices": [
    { "amount": 4999, "currency": "INR" },
    { "amount": 999, "currency": "USD" }
  ],
  "features": ["Unlimited users", "Priority support"],
  "metadata": {
    "tier": "professional",
    "version": "2"
  },
  "created_at": "2024-01-15T10:30:00Z",
  "updated_at": "2024-03-10T14:22:00Z"

```

<Note>
Updating a plan does not affect existing subscriptions. Changes only apply to new subscriptions created after the update.
</Note>
