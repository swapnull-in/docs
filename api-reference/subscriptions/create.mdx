---
title: "Create Subscription"
openapi: "POST /v1/subscriptions"
description: "Create a new subscription"
---

## Request Body

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `customer_id` | string | Yes | Customer ID |
| `plan_id` | string | Yes | Plan ID |
| `quantity` | integer | No | Quantity (for per-seat pricing) |
| `trial_end` | string | No | Trial end date (ISO 8601) |
| `coupon` | string | No | Coupon code to apply |
| `payment_gateway` | string | No | `razorpay` or `stripe` |
| `metadata` | object | No | Custom key-value data |

## Example Request

```bash
curl -X POST https://api.recurso.dev/v1/subscriptions \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "customer_id": "cust_abc123",
    "plan_id": "plan_pro",
    "payment_gateway": "razorpay"
  }'
```

## Response

```json
{
  "id": "sub_xyz789",
  "object": "subscription",
  "customer_id": "cust_abc123",
  "plan_id": "plan_pro",
  "status": "created",
  "quantity": 1,
  "current_period_start": "2024-01-15T00:00:00Z",
  "current_period_end": "2024-02-15T00:00:00Z",
  "cancel_at_period_end": false,
  "checkout_url": "https://pay.recurso.dev/checkout/sub_xyz789",
  "created_at": "2024-01-15T10:30:00Z"
}
```

<Info>
Redirect the customer to `checkout_url` to complete payment.
</Info>
