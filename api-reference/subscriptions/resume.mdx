---
title: "Resume Subscription"
openapi: "POST /v1/subscriptions/{id}/resume"
description: "Resume a paused subscription"
---

## Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | The subscription ID |

## Example Request

```bash
curl -X POST https://billing.example.com/v1/subscriptions/sub_xyz789/resume \
  -H "Authorization: Bearer $API_KEY"
```

  "current_period_end": "2024-06-01T00:00:00Z",
  "paused_at": null,

<Note>
Resuming a subscription starts a new billing period immediately. An invoice is generated for the upcoming period.
</Note>
